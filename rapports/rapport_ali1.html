Divergence de Kullback-Leibler explication des poids

La divergence de Kullback-Leibler (KL) est une mesure statistique utilise pour quantifier la diffrence entre deux distributions de probabilits. Elle est souvent utilise en thorie de l'information, en statistique et en apprentissage automatique pour mesurer combien une distribution ( P ) s'carte d'une distribution de rfrence ( Q ). Voici une explication dtaille de la divergence KL et de son utilisation des poids dans le contexte du traitement de squences biologiques.

La divergence KL de la distribution ( P ) par rapport la distribution ( Q ) (note ( D_{KL}(P \parallel Q) )) est dfinie comme suit :

[ D_{KL}(P \parallel Q) = \sum_{i} P(i) \log \left(\frac{P(i)}{Q(i)}\right) ]

o ( P ) et ( Q ) sont des distributions de probabilits, et la somme est prise sur tous les vnements ( i ) possibles dans ces distributions. Voici quelques points cls propos de la divergence KL :

  • Mesure non symtrique : ( D_{KL}(P \parallel Q) ) n'est pas gale ( D_{KL}(Q \parallel P) ). Cela signifie que le "cot" de passer de ( P ) ( Q ) est diffrent de celui de passer de ( Q ) ( P ).

  • Interprtation : Si ( P ) est identique ( Q ), la divergence KL est zro car il n'y a aucune diffrence entre les deux distributions. Des valeurs plus leves indiquent des diffrences plus grandes entre ( P ) et ( Q ).

  • Domaine de valeurs : La divergence KL est toujours non-ngative (( D_{KL}(P \parallel Q) \geq 0 )), avec galit seulement quand ( P ) est identique ( Q ).

Poids associs la divergence KL

  1. Calcul des distributions : Pour chaque position dans l'alignement, calculez la distribution des acides amins pour chaque ensemble de squences (par exemple, deux MSA diffrents).

  2. Divergence KL pour chaque position : Calculez la divergence KL entre les deux distributions chaque position. Cela donne une mesure de combien les distributions diffrent cette position prcise.

  3. Attribution de poids : Utilisez la divergence KL comme un "poids" pour cette position. Des valeurs plus leves indiquent des diffrences plus marques et, potentiellement, une importance biologique plus grande. Ces poids peuvent influencer des analyses ultrieures, comme le calcul de matrices de prsence ou l'ajout de squences un MSA.

  4. Normalisation : Les poids peuvent tre normaliss de manire ce que leur somme totale soit constante, permettant une comparaison uniforme travers l'alignement.
